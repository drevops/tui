# Example project

A small example project the playground demos read from - no real data, just enough structure to exercise two features. The file-picker demos (`02-widgets-filepicker*.php`) browse it for files: price lists (`baskets/*.csv`, `harvest.csv`), the `pantry.yaml` stock reference and a `deliveries/` log. The discovery demo (`08-discovery.php`) scans it for update-mode defaults: `box.json`, `.env` and the `baskets/` subdirectories.
