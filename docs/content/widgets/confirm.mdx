# Confirm

<p align="center">
  <ThemedImage sources={{light: useBaseUrl('/widget-confirm-light-animated.svg'), dark: useBaseUrl('/widget-confirm-dark-animated.svg')}} alt="Confirm widget" />
</p>

Yes/No toggle. Arrows or Space switch, `y`/`n` set the choice directly, Enter
accepts.

```php
$p->confirm('cdn', 'Serve via CDN?')->default(TRUE);
```
