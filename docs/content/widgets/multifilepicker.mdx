# MultiFilePicker

Like the file picker, but several paths can be chosen: `Space` toggles the
highlighted entry and selections accumulate as you browse between directories,
`Enter` accepts them all. It collects the list of chosen paths.

```php
$p->multiFilePicker('assets', 'Asset files')->startIn(getcwd())->extensions(['png', 'svg']);
```
