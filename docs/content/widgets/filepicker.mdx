# FilePicker

Browse the filesystem for a single path. Arrows move, `→` enters the highlighted
directory and `←` returns to its parent (never above the start directory), typing
filters the current directory and `Tab` reveals hidden entries; `Enter` selects
the highlighted entry. Restrict what may be chosen with `filesOnly()` or
`directoriesOnly()`, limit files with `extensions()`, and choose where browsing
starts with `startIn()`. It collects the chosen path.

```php
$p->filePicker('config', 'Config file')->startIn(getcwd())->filesOnly()->extensions(['yml', 'yaml']);
```
