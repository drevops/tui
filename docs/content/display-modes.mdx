# Display modes

The TUI adapts to the terminal it lands on: glyphs follow the locale (a `UTF` locale in `LC_ALL`, `LC_CTYPE` or `LANG` enables Unicode; anything else falls back to ASCII) and color honors [`NO_COLOR`](https://no-color.org/) and `TERM=dumb`. You can force either on the `Tui` facade with `->unicode(TRUE|FALSE)` and `->color(TRUE|FALSE)`. Here's the same produce-box form in each combination:
